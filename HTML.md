# 《HTML入门笔记1》

1. HTML是Tim Berners-Lee(李爵士)发明的。
2. HTML起手：
   ```
      <!DOCTYPE html>
      <!-- en 可更换为 zh-CN -->
      <html lang="en">
      <head>
          <meta charset="UTF-8">
          <!-- IE使用最新内核 -->
          <meta http-equiv="X-UA-Compatible" content="IE=edge">
           <!-- 禁用缩放-->
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Document</title>
      </head>
      <body>

      </body>
      </html>
    ```

3. 表章节的标签
    * h1~h6       标题
    * section     章节
    * article     文章
    * p           段落
    * header      头部
    * footer      脚步
    * main        主要内容
    * aside       旁支内容
    * div         划分
